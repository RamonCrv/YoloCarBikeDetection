API

  • RESTful
  • Grátis
  • Tamanho Máximo do Vídeo: 100MB
  • Extensões suportadas: .avi .mp4 .MOV
  • URL: http://127.0.0.1:8000/analyze/
  • Enviar solicitação via método POST com o vídeo para o corpo com o nome 'video'
  • Exemplo de Retorno:
